  - 名前入力欄
  - ユーザID入力欄
  - パスワード入力欄

  - サイト名

 - 検索窓画面

  - カテゴリ選択欄
  - クライアント名

  - サイト名

  - 本の情報

  - もどるボタン
  - クライアント名

  - サイト名
  - 基本機能[TOPボタン、カートボタン、ログアウト]
  - カート内の本の情報
  - カート内から本を削除ボタン(個別、全削除)
  - クライアント名
